# Test Project Documentation

This directory contains documentation for the test project.

## Structure

- `api/` - API service with Dockerfile
- `backend/` - Backend service with Dockerfile  
- `frontend/` - Frontend service with Dockerfile
- `shared/` - Shared components with Dockerfile
- `library/` - Pure code library (no Dockerfile)
- `utils/` - Utility modules (no Dockerfile)

## Purpose

This project is used to test Dockerz behavior with different scenarios.
